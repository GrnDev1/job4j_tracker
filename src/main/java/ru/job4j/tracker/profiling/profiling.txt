1. При помощи функции
jps
получаем pid (2468);
2. При помощи функции
jmap -histo 2468
получаем память о классе:
порядковый номер, количество экземпляров класса, количество байт, имя класса;
3. При помощи функции
jhsdb jmap --heap --pid 2468
получаем информацию о куче:
young generation, old generation, regions...;
4. При добавлении огромного количества заявок получил exception:
java.lang.OutOfMemoryError
